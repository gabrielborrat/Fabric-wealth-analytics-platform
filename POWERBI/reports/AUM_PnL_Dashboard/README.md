# AUM & P&L Dashboard

## Objectives
This dashboard provides comprehensive views of Assets Under Management (AUM) and Profit & Loss (P&L) metrics.

## Pages
1. **Overview**: High-level AUM and P&L summary
2. **AUM Detail**: Detailed breakdown of AUM by client, account, and security
3. **P&L Detail**: Detailed P&L analysis with realized and unrealized components

## Target Audience
- Portfolio managers
- Risk managers
- Executive leadership
- Client relationship managers

## Key Metrics
- Total AUM
- AUM by client
- Daily P&L
- Realized vs Unrealized P&L
- AUM growth trends
