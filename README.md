# Fabric Wealth Management Analytics Platform

This repository contains the complete implementation of a Wealth Management Analytics solution built on Microsoft Fabric.

## Overview

This platform provides end-to-end analytics capabilities for wealth management, including:
- Data ingestion from multiple sources (FX rates, transactions)
- Data transformation through Bronze, Silver, and Gold layers
- Data warehouse implementation with star schema
- Power BI dashboards for reporting and analytics

## Structure

- `docs/` - Documentation including architecture diagrams, portfolio, screenshots, and data dictionaries
- `src/` - Source code organized by layer (ingestion, silver, gold, warehouse)
- `powerbi/` - Power BI semantic models, reports, and datasets
- `pipelines/` - Fabric pipeline definitions
- `governance/` - Governance documentation and policies
